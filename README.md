This project evaluates the robustness and calibration performance of Convolutional Neural Networks (CNNs) and Vision Transformers (ViTs) under human vision-motivated degradations. 

We apply four types of human-vision–motivated degradations: foveated blur, contrast loss, shot noise, and fixation jitter. These degradations cover five severity levels, ranging from normal vision to the limits of human perception. 

Using 5,000 images from the ImageNet validation set, we evaluate multiple pre-trained architectures across accuracy and Expected Calibration Error (ECE).
